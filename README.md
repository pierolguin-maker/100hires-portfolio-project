## Challenges and Solutions
While setting up my GitHub account, I had some issues with the browser that didn’t let me complete the process normally. I solved it by using an incognito window, which allowed me to finish the setup.

When trying to use Git on my Mac, I ran into an issue because the necessary developer tools were not installed. I fixed this by installing Xcode Command Line Tools.

I also had some confusion connecting my local project with GitHub, but I was able to solve it step by step and eventually get everything working.

## What I Learned
This project helped me understand how Git and GitHub work together, and how to troubleshoot common setup issues. It also showed me the importance of being patient, trying different solutions, and looking things up when something doesn’t work at first.
